CSV Format Guide

The first row must be the header. Required columns:

Column Required Example Notes
type✅locallocal = govt bus, private = private operator
bus_number✅TN72A0001Any unique identifier
operator✅TNSTCFor private buses, this is the company name
origin✅SrivilliputhurDeparture city
destination✅MaduraiArrival city
departure_time✅06:0024-hour format HH:MM
arrival_time✅08:3024-hour format HH:MM
fare✅80In Rupees
bus_typeExpressOrdinary / Express / AC Sleeper etc.
statusscheduledscheduled / running / delayed / available
via_stopsSalem;TrichySemicolon-separated stop names
route_numberSVP-001For govt buses only
route_nameMadurai – ChennaiFor private buses only
rating4.5Private buses only (0–5)
live_trackingyesyes / no – for private buses
duration7h 30mJourney duration (text)
seat_availability40Available seats
total_seats50Total seat capacity
Sample row (local):
local,TN72A0001,TNSTC,Srivilliputhur,Madurai,06:00,08:30,80,Express,scheduled,Virudhunagar,SVP-001,,0,no,,50,50

Sample row (private):
private,KA01P001,KPN Travels,Madurai,Chennai,21:00,07:00,600,AC Sleeper,available,Salem;Trichy,,Madurai - Chennai,4.5,yes,10h,40,40
